Also, not be excluded, was the idea that perhaps one day, I could finally obtain that elusive degree as well as obtain some IT skills I currently don’t have such as application development.

Project Idea

Overview

This project will be to design and develop an online platform, complete with mobile apps, that will connect owners of goats with owners of land that need clearing of thick undergrowth and scrub. The platform will benefit both goat herd owners as well as landowners.

As a goat owner, the platform registration process will require information to be entered including:

  • Location (address) of area to be cleared
  • Approximate area (in hectares and acres) of the land
  • Details of fencing of the area (fully, partially, not fenced)
  • Details of the topography of the land (flat, undulating, steep, rocky, etc)

Motivation

Currently in Australia, there are around 7 million people living in rural parts of the country (Australian Institute of Health and Welfare 2018. Australia’s health 2018. Viewed 12 September 2019, . These properties typically require maintenance to control scrub, weeds and undergrowth that is undesirable. Typically, reduction of this vegetation required landowners to resort to one of the following methods.

  • Manual removal by hand
  • Manual removal using machinery
  • Herbicides using Glyphosate

While each of these methods are effective in the short term, they all pose their own drawbacks across several areas.

Manual or machine assisted removal is both laborious and time consuming which means it is also expensive. Additionally, the removal is only temporary and must be repeated several times annually as the unwanted vegetation typically returns rapidly.

Poisoning of vegetation can be less time consuming and expensive but can pose a serious threat to both the environment and animal life. Glyphosate, the active ingredient in many herbicide poisons, has been banned in 18 countries globally (Sustainable Pulse. Glyphosate Herbicides Now Banned or Restricted in 18 Countries Worldwide – Sustainable Pulse Research May 2019, viewed Sept 12 2019< https://sustainablepulse.com/2019/05/28/glyphosate-herbicides-now-banned-or-restricted-in-17-countries-worldwide-sustainable-pulse-research/#.XX2lsSgzaUk>. While still legally used in Australia, many municipalities are currently testing alternate methods of weed control after the World Health Organisation warned that Glyphosate probably causes cancer Scientific American March 2015, Widely Used Herbicide Linked to Cancer, viewed Sept 13 2019, < https://www.scientificamerican.com/article/widely-used-herbicide-linked-to-cancer/>.

Business Model Description

The solution will connect both goat owners and landowners and would be designed such that landowners are charged a fee, based on a calculation that factors:

  • Area to be cleared
  • Time required to clear (based on number of goats in herd)

The fee would represent the bulk of revenue generated by the platform, while ensuring the fee paid by landowners would be a significant saving when compared to manual or machine assisted clearing. The fee would be comparable to poisoning however, would be significantly more environmentally safe.

The goat owners would pay a small, annual subscription fee to use that platform, but would receive a significant proportion of the landowner’s fee for their goats to be placed on the land. Additionally, the goat owners receive the benefit of zero cost of feed for the duration of the placement.

Features

The solution comprises of a web application that can be accessed via any online browser. Additionally, the users will be able to download a mobile app that will interface with the web application providing a subset of features provided by the web portal.

On fist visiting the website, users will be presented a “two halves” style of presentation. It will be clear to the user that they need to either navigate to the Goat Owners section or the Landowner Section.

Goat Owners Section

This Section of the Web Application Includes:

  • Descriptive summary of benefits to the goat owner as well as the environment
  • Description of the business model and monetary benefit to goat owners
  • A “Register Now” function that collects all necessary goat owner’s information

Landowners Section

This Section of the Web Application Includes:

  • Descriptive summary of benefits to the landowner as well as the environment
  • Description of the business model and costs to landowners
  • A “Register Now” function that collects all necessary landowner’s information

Database

The system, having collected all the relevant information from both parties, will populate a live database. The DB will interface into a geolocation service that will calculate proximity of each party and then present a suitable match.

Transaction

Once a match between two parties is established, each party will be presented with an Accept action. If both parties action this in the affirmative, then a match is confirmed, and the transaction is initiated. Both parties are notified of the transaction and contact details for both parties are made available to the other.

Tools and Technologies

This project idea is not particularly ground-breaking from a technology perspective, rather, it is unique in that it takes a model commonly used in the short-term property letting industry and applies it differently to solve some problems not previously solved. As such, we can lean on these projects and borrow from the lessons they learned in relation to technology. However, to keep our costs down, we will opt for open source technology wherever possible.

As the project is essentially a web application, technology can be broken down into following categories with suggested technologies.

  • Web Application Framework – Ruby on Rails (Open Source)
  • SQL Data Storage – PostgreSQL (Open Source)
  • Web Server – Nginx (Open Source)
  • Application Server – Puma (Open Source)
  • Hosting – Amazon Web Services (AWS) Commercial

Skills Required

Skills required to execute this project are varied. But, with open source technology forming the core of our solution, talent should be readily available.

Some of the skills required will include:

  • Programming languages — Ruby, JavaScript, HTML5, CSS3 skills
  • Infrastructure Engineering – AWS Engineering skills

Outcome

If this project were to be successful, the outcome with the greatest impact would be a noticeable reduction in the use of harmful pesticides in Australia. The benefits of this are obvious.

Additionally, the outcomes for both parties are also worth noting. For landowners, an environmentally friendly method of week control that is cheaper than conventional methods.

For the goat owner, an additional income stream for a demographic that has been doing it tough of late with the onset of climate change and drought. Additionally, a financial saving with reduced need to purchase livestock feed.

For a Solutions Architect, both these areas are of equal importance and provide a career satisfaction double that of a career in just one of the areas.
